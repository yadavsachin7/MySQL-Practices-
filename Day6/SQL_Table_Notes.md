# SQL Day 6 Notes

## Concepts Covered

- `COUNT()` → Count rows
- `SUM()` → Total value
- `AVG()` → Average value
- `MAX()` → Highest value
- `MIN()` → Lowest value
- `GROUP BY` → Group similar data
- `HAVING` → Filter grouped data
- `ORDER BY` → Sort data
- `LIMIT` → Display limited rows

# 📌 Concepts Used

- Aggregate functions are used to calculate values like count, sum, average, maximum, and minimum.
- `GROUP BY` groups similar records, while `HAVING` filters those groups.
- `ORDER BY` sorts the output, and `LIMIT` restricts the number of rows returned.
