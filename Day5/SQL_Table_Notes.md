## Interview Notes

- `UPDATE` modifies existing records.
- `DELETE` removes selected rows using `WHERE`.
- `ALTER TABLE ADD` is used to add a new column.
- `TRUNCATE TABLE` removes all records and resets `AUTO_INCREMENT`.
- Always use the `WHERE` clause with `UPDATE` and `DELETE` to avoid affecting all rows.
- `TRUNCATE` is faster than `DELETE` because it removes all rows at once.
